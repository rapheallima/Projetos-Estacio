## Como executar

1. Abra o terminal na pasta onde está o arquivo `super_trunfo.c`.

2. Compile o programa com o comando:

```bash
gcc super_trunfo.c -o super_trunfo.exe

````

3. Execute o programa com o comando:

```bash
./super_trunfo.exe
